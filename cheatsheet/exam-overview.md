# Google Cloud Associate Cloud Engineer - Exam Overview

## üìã Exam Information

### Basic Details
- **Exam Name:** Associate Cloud Engineer
- **Duration:** 2 hours
- **Number of Questions:** Approximately 50 questions
- **Question Format:** Multiple choice and multiple select
- **Registration Fee:** $125 USD
- **Certification Validity:** 3 years from the date you pass
- **Prerequisites:** None (but hands-on experience recommended)
- **Recommended Experience:** 6+ months of hands-on experience with GCP

### Exam Domains and Weightage

#### 1. Setting up a cloud solution environment (17.5%)
- Setting up cloud projects and accounts
- Managing billing configuration
- Installing and configuring the Cloud SDK

**Key Topics:**
- Creating projects with Cloud Console and Cloud SDK
- Assigning users to predefined IAM roles
- Setting up billing accounts and linking projects
- Installing and using Cloud SDK (gcloud, gsutil, bq)
- Configuring Cloud Shell

#### 2. Planning and configuring a cloud solution (17.5%)
- Planning and estimating GCP product use using the Pricing Calculator
- Planning and configuring compute resources
- Planning and configuring data storage options
- Planning and configuring network resources

**Key Topics:**
- Compute Engine, GKE, Cloud Run, Cloud Functions
- Cloud SQL, Cloud Spanner, Firestore, Bigtable
- Cloud Storage classes and lifecycle management
- VPC, Subnets, Firewall rules, Load Balancing

#### 3. Deploying and implementing a cloud solution (25%)
- Deploying and implementing Compute Engine resources
- Deploying and implementing Google Kubernetes Engine resources
- Deploying and implementing Cloud Run and Cloud Functions resources
- Deploying and implementing data solutions
- Deploying and implementing networking resources
- Deploying a solution using Cloud Marketplace
- Implementing resources via infrastructure as code

**Key Topics:**
- VM instances, instance templates, instance groups
- GKE clusters, deployments, services
- Cloud Functions triggers and Cloud Run services
- Cloud SQL, Datastore, BigQuery, Cloud Storage
- VPC, firewall rules, VPN, Cloud DNS
- Cloud Deployment Manager, Terraform

#### 4. Ensuring successful operation of a cloud solution (17.5%)
- Managing Compute Engine resources
- Managing Google Kubernetes Engine resources
- Managing Cloud Run resources
- Managing storage and database solutions
- Managing networking resources
- Monitoring and logging

**Key Topics:**
- Starting, stopping, editing VMs and instances
- SSH into instances, viewing logs
- Managing GKE nodes and pods
- Object lifecycle management in Cloud Storage
- Cloud Monitoring and Cloud Logging
- Creating alerts and dashboards

#### 5. Configuring access and security (22.5%)
- Managing Identity and Access Management (IAM)
- Managing service accounts
- Viewing audit logs

**Key Topics:**
- IAM roles (primitive, predefined, custom)
- Service accounts creation and management
- IAM policies at project and resource level
- Cloud Audit Logs
- Security best practices

## üìù Exam Registration

1. Visit the [Google Cloud Certification website](https://cloud.google.com/certification/cloud-engineer)
2. Create or sign in to your Webassessor account
3. Select "Register for an Exam"
4. Choose between online proctored or test center
5. Pay the exam fee ($125 USD)
6. Schedule your exam

## üèÜ After Passing

Once you pass:
- You'll receive a digital certificate and badge
- Your certification will be valid for 3 years
- You can share your achievement on LinkedIn
- You'll be listed in the Google Cloud Certification directory
- You'll receive promotional items from Google
