## üéØ Exam Domains

The Associate Cloud Engineer exam covers the following domains:

1. **Setting up a cloud solution environment** (17.5%)
2. **Planning and configuring a cloud solution** (17.5%)
3. **Deploying and implementing a cloud solution** (25%)
4. **Ensuring successful operation of a cloud solution** (17.5%)
5. **Configuring access and security** (22.5%)
